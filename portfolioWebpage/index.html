HTML Tags and Elements
Before we go further, I will show tags and elements essential to a begginer.
  • Root element
    • Web pages start with the HTML element - the root element. The root element is responsible to encloses all the other elements, being the sole parent element to all the others. To create the html element, you write an opening < HTML > tag followed by a closing < / HTML > . You can check more information from W3 school.
  • Documentation head
    • Head element contains informations about web page such as:
    • Title: Will be explained below;
    • Link: To add style sheets and favicons into your page;
    • Meta: To specify things as character sets, page descriptions and keywords for search engines;
    • Script: To add JavaScript code into the page.
    More informations about documentation head you can MDN Web docs
  • Page Title
    • The element that contains the title of the page to be displayed in the browser's title bar.
  • Body
    • This element appears after the head element and contain all the content of the web application.
  • Section Heading
    • HTML supports 6 headings elements: h1, h2, h3, h4, h5, and h6. The most important is h1, h2 is for less important subheadings and so on.
  • Paragraph
    • The Paragraph element - detoted by p - create a paragraphs of text.
  • Link
    • An important element in web page is the link element - denoted by a. It is important to give links meaning by using descriptive link text. Also, make links navigable with HTML acess keys attribute to specify a shortcut key to activate or bring focus to an element. You can check more about this important element in the freeCodeCamp course
  • Image
    • This element let you insert images into your applications. From freeCodeCamp I learned that is important to add alternative text in images for visually impaired accessibility.
  • Block-Level
    • Div element is a generic container to add more structure to your application. You use, for example, to group several paragraphs or headings with similar purpose together inside it.
  • Inline container
    • Span element is quite similar to div, but it is an inline element instead.
